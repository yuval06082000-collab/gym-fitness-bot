# ST-FITNESS Chat Bot Project 🏋️‍♂️

מערכת צ'אט חכמה לחדר כושר המאפשרת למשתמשים להתכתב עם מאמן כושר וירטואלי בשם "בן". הפרויקט כולל ממשק CLI (טרמינל) וממשק אינטראקטיבי מתקדם מבוסס Streamlit בעיצוב מותאם אישית בסגנון WhatsApp.

## 📋 מבנה הפרויקט
* `script_cli.py` - ממשק צ'אט בסיסי הרץ בטרמינל 
* `script12.py` - ממשק צ'אט ויזואלי מתקדם מבוסס Streamlit 
* `requirements.txt` - רשימת הספריות והתלויות של הפרויקט
* `.gitignore` - הגדרות החרגה לשמירה על אבטחת מפתחות ה-API וקבצי מערכת
